# assets/

Drop your media here.

- **og.jpg** — 1200×630 social share image (shown when the link is pasted
  into Instagram DMs, iMessage, etc.). Referenced in `index.html`.
- **logo.svg / logo.png** — your wordmark, if you make one.
- **crew photos** — to use a real photo on a crew card, add it here and set
  `img: "assets/yourphoto.jpg"` on that member in `js/config.js`.
- **event flyers / recap stills** — keep them here, reference from `index.html`
  if you want to feature them.

Keep files reasonably small (compress images) so the site stays fast on phones.
